Rationale

"Hopeless Fountain Kingdom"
By Halsey
This project provides an interesting, although not always easy, opportunity for exponential learning. It seemed very daunting at first but became a welcoming weakly challenge where I tried to put myself to learn more every week. I tried to encapsulate a very duel nature in everything I did in this project to encapsulate the same message that Halsey frequentely does, like using the imagery of a bee for sweetness or contrasting the very old fashioned serif cursive font with a modern sans-serif font and contrasting more liquid movement with fast rigid movement. I did this to try and emulate how Halsey tries to put this message across to express her struggles with bipolar, a common theme in her music, and raise awareness for it. This project has taught me so much and has opened my eyes to a field that I would genuinely consider going into one day. I have thoroughly enjoyed it and look forward to bringing some aspects of it forwards with me to theme 4s project.
